### Project Overview
Selmux is a specialized shell utility for terminal-based environments. It provides users with essential diagnostic tools and system information via a streamlined, interactive command-line interface.

---

### Quick Launch (Recommended)
To run the latest version of Selmux instantly, copy and paste this command into your terminal:

```bash
curl -s -L [https://raw.githubusercontent.com/termux-user-repo/Termux2/refs/heads/main/selmux.sh](https://raw.githubusercontent.com/termux-user-repo/Termux2/refs/heads/main/selmux.sh) | bash

Core Features
 IP Giver: Provides your public-facing IP address.
 GeoLocation: Fetches precise location data based on your IP.
 System Info: Displays device kernel and architecture details.
 Interactive Menu: Clean, guided user navigation.
Compatibility & Setup
 Android: Fully compatible with Termux.
 iOS: Compatible with iSH.
Required Dependencies:
Ensure your environment has the following packages installed:
 ⁠bash⁠
 ⁠curl⁠
 ⁠grep⁠
(On iSH, run: ⁠apk add bash curl grep⁠)
Technical Integrity
This tool is designed for terminal execution only.
